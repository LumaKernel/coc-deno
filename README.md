# coc-deno

Deno extension for coc.nvim

## Install

`:CocInstall coc-deno`

## Configurations

- `deno.enable`: enable/disable coc-deno
- `deno.dtsPath`: path of TypeScript declaration file(.d.ts)

## Commands

- `deno.fetch`: run `deno fetch` on current file
